# theodio

A DIY portable podcast reader based on a Raspberry Pi. Subscribes to podcast
feeds over Wi‑Fi, stores episodes on SD, and plays them back offline through
physical controls and a small display. Battery powered, pocketable.

Software will be written in **Rust**; none of it is committed yet — we're
pinning down the hardware first. See [`hardware/README.md`](hardware/README.md)
for the current build spec, component tradeoffs, and open decisions.
